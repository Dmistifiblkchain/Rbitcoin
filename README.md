# Rbitcoin

Utilities related to Bitcoin. Unified markets API interface (bitstamp, kraken, btce, bitmarket). Both public and private API calls.

## Installation

```R
install.packages("Rbitcoin")
```

## Usage

```R
library(Rbitcoin)
?btc
```
